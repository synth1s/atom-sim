## Anti-Patterns (nao repetir)

1. **NAO criar mesh/material por frame.** Sempre pre-alocar em Resource no setup. (Corrigido em commit 9d53945)
2. **NAO usar queries sem filtro de era.** collide_particles afeta TODAS as entidades — deve filtrar por marker. (Problema conhecido, ainda nao corrigido)
3. **NAO esquecer cleanup de Resources.** Todo Resource inserido no setup deve ser removido no cleanup.
4. **NAO usar EventReader.** Em Bevy 0.18 e MessageReader.
5. **NAO usar .single() sem tratar Result.** Sempre `let Ok(x) = query.single() else { return };`
